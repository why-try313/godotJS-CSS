Install

  1. Clone the repository in your project res://addons/ folder: git clone https://github.com/why-try313/godotJS-CSS.git
  2. Enable the addon in Godot Project > Project Settings... > Plugins

Usage

By creating a res://css/style.css file:

  1. Create a style.css file in res://css/style.css
  2. Insert a Div element in your scene: Add Child Node > Node > CanvasItem > Panel > Div
  3. Set class or id property on your Div element

With inline CSS

  1. Insert a Div element in your scene: Add Child Node > Node > CanvasItem > Panel > Div
  2. Set inline CSS property on your Div element using self { ...properties } as element (self:hover {} and self:active {} for mouse events)

Note: you can combine class/id values with inline CSS properties (inline CSS override CSS values)
